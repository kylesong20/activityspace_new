# activityspace

#### 介绍
高校社团活动场地管理系统的设计与实现-毕业设计作品


#### 部署前提
1.  Redis-x64-5.0.14.1
2.  Nacos 2.2.0

#### 使用说明
1.  activityspace_new 是整个项目代码，运行前请先下载项目依赖
2.  vue1002 目录为管理员用户网站前端，运行前请先下载项目依赖
3.  activityspace_front 目录为普通用户前台app,运行请使用Hbuilder启动，运行前请先下载项目依赖
